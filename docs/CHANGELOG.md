## [1.0.1] - 2024-03-27

### 部署改进
- 📦 新增 musl 静态编译支持，解决 GLIBC 版本兼容性问题
- 📖 更新文档，添加 CentOS 7.6 等旧系统部署指南

## [1.0.0] - 2024-03-27

### 核心变更
- 🦀 完全重写为 Rust 实现（从 Python/Bash 迁移）
- ⚡ 性能提升 10 倍，内存减少 90%
- 📦 单一二进制文件部署

### 主要功能
- 实时流量监控（每秒采样）
- 自动发现网络命名空间
- 三级数据聚合（1s/10s/1m）
- REST API + SSE 实时推送
- Web 可视化界面

### 技术栈
- Actix Web (HTTP 服务器)
- Tokio (异步运行时)
- SQLite (数据存储)
- 100% Rust 实现

### 性能指标
- 内存: 50-100 MB (300 命名空间)
- 延迟: 5-50ms
- 并发: 10,000+ 连接
- 吞吐: 10,000+ req/s
